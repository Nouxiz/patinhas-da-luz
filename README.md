# Patinhas da Luz 🐾

Aplicação web desenvolvida como projeto acadêmico com foco em integração entre front-end e back-end, utilizando persistência de dados em banco relacional.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL (executado via XAMPP)
- Apache Server

## 📌 Funcionalidades Implementadas

- Cadastro de informações com persistência em banco de dados
- Consulta e exibição dinâmica de registros
- Integração entre interface do usuário e backend
- Manipulação de dados através de queries SQL (INSERT e SELECT)


## 📈 Status do Projeto

Projeto acadêmico desenvolvido para fins de aprendizado e aprimoramento técnico.
